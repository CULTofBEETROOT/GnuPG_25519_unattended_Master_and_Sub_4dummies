# GnuPG 25519 Unattended Master and Subkeys Generator
# & USB-Luks (gpg-key storage) dongle-Formater.

مرور کلی
این مخزن مجموعه‌ای از اسکریپت‌ها را برای تولید GPG Master و Subkeys با استفاده از یک کلید USB با رمزگذاری LUKS ارائه می‌دهد. این اسکریپت‌ها برای کاربران غیر روت طراحی شده‌اند اما برای برخی عملیات به دسترسی روت نیاز دارند. این تنظیمات برای Debian Trixie طراحی شده است و برای کاربرانی در نظر گرفته شده است که ممکن است در طول فرآیند روی سخت‌افزارهای قدیمی‌تر و بدون دسترسی به اینترنت کار کنند.

نکات مهم
* ترتیب اجرا: اسکریپت‌ها باید به ترتیب مشخص شده اجرا شوند: اول، دوم و سوم. هیچ مرحله‌ای را نادیده نگیرید، حتی اگر یک مرحله با شکست مواجه شود.

* محیط: قبل از شروع، مطمئن شوید که فایل‌های لازم را دارید:

FIRST.sh

SECOND.sh

THIRD.sh

(به صورت اختیاری: batchproduction_ofTHIRD.sh)

...در دایرکتوری دانلودهای شما.


# برای تولید دسته‌ای (بهتر!)

* اگر چندین آدرس ایمیل دارید که می‌خواهید برای آنها کلیدهای GPG تولید کنید، دستورات زیر را با کاربر root یا با sudo اجرا کنید:

```bash
sudo bash /home/$USER/Downloads/FIRST.sh;
sudo bash /home/$USER/Downloads/SECOND.sh;
```

* دو فایل ایجاد کنید:

یکی با ستونی از مالکان،

دومی با ستونی از آدرس‌های ایمیل

و آنها را با زیرعنوان فیلدهای آرگومان در الگو تغذیه کنید:


```bash
/home/$USER/Downloads/batchproduction_ofTHIRD.sh <path/to/and/ownerfile.txt> <path/to/and/addressfile.txt>;
```

این اسکریپت آخر به شما امکان می‌دهد لیست ایمیل‌ها و مالکان مربوطه را وارد کنید و اسکریپت‌های bash جداگانه‌ای برای هر آدرس ایمیل تولید کنید.

* وقتی اسکریپت‌های جداگانه تولید شده را اجرا می‌کنید، کلیدهای GPG ذخیره شده در کلید USB رمزگذاری شده با LUKS شما ایجاد می‌شوند. مهم: OpenGPG توصیه می‌کند که فوراً از کلیدهای خصوصی رمزگذاری نشده خود استفاده کرده و سپس آنها را حذف کنید تا از ذخیره اطلاعات حساس جلوگیری شود.

# یا تولید تک کلید را انتخاب کنید (به صورت شهودی).

مراحل انجام کار

مرحله 1: اجرای اسکریپت اول

1. یک ترمینال باز کنید. ۲. دستور زیر را به عنوان کاربر ریشه یا با sudo اجرا کنید:

```bash
 bash /home/$USER/Downloads/FIRST.sh
```

مرحله ۲: ایجاد کلید USB با LUKS
۱. این مرحله را در حالت ریشه شروع کنید:

```bash
 bash /home/$USER/Downloads/SECOND.sh
```


مرحله ۳: ذخیره GPG Master و Subkeys در دستگاه USB
۱. این مرحله را در حالت ریشه شروع کنید. ممکن است لازم باشد مجوزهای اسکریپت را تغییر دهید تا قابل اجرا شود.
از دستور زیر استفاده کنید:

```bash
 chmod +x /home/$USER/Downloads/THIRD.sh
```


۲. سپس، اسکریپت را اجرا کنید:

```bash
 /home/$USER/Downloads/THIRD.sh
```

مرحله نهایی: جدا کردن ایمن دستگاه USB

پس از اتمام موفقیت‌آمیز همه مراحل، مطمئن شوید که دستگاه USB LUKS خود را به طور ایمن جدا کرده‌اید تا از مشکلات احتمالی هنگام اتصال مجدد آن جلوگیری کنید:

```bash
sudo umount "/mnt/usb_gpg"
sudo cryptsetup luksClose myusb_key
```

در غیر این صورت دفعه بعد که دستگاه خود را وصل می‌کنید، احتمالاً با مشکلاتی مواجه خواهید شد.

نتیجه‌گیری

با دنبال کردن این مراحل، می‌توانید GPG Master و Subkeys خود را به طور ایمن روی یک کلید USB رمزگذاری شده با LUKS تولید و ذخیره کنید. همیشه به یاد داشته باشید که کلیدهای خصوصی خود را با احتیاط مدیریت کنید و بهترین شیوه‌های امنیتی را دنبال کنید. در صورت داشتن هرگونه سوال یا مشکلی، لطفاً به مستندات مراجعه کنید یا از انجمن کمک بگیرید.
