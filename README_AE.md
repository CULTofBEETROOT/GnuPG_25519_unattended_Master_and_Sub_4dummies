# مولد مفاتيح رئيسية وفرعية تلقائي لـ GnuPG 25519
# و USB-Luks (تخزين مفاتيح GPG) - مُهيئ دونجل.

نظرة عامة
يوفر هذا المستودع مجموعة من البرامج النصية لإنشاء مفاتيح GPG رئيسية وفرعية باستخدام مفتاح USB مع تشفير LUKS. صُممت هذه البرامج النصية للمستخدمين غير الجذر، ولكنها تتطلب صلاحيات الجذر لبعض العمليات. هذا الإعداد مُخصص لنظام Debian Trixie، وهو مُوجه للمستخدمين الذين قد يعملون على أجهزة قديمة بدون اتصال بالإنترنت أثناء العملية.

ملاحظات هامة

* ترتيب التنفيذ: يجب تشغيل البرامج النصية بالترتيب المحدد: الأول، الثاني، والثالث. لا تتجاوز أي خطوة، حتى في حالة فشل إحداها.

* بيئة التشغيل: تأكد من وجود الملفات اللازمة:

FIRST.sh

SECOND.sh

THIRD.sh

(اختياري: batchproduction_ofTHIRD.sh)

...في مجلد التنزيلات قبل البدء.

# للإنتاج الدفعي (أفضل!)

* إذا كان لديك عدة عناوين بريد إلكتروني ترغب في إنشاء مفاتيح GPG لها، فقم بتشغيل الأوامر التالية بصلاحيات المستخدم الجذر أو باستخدام sudo:

```bash
sudo bash /home/$USER/Downloads/FIRST.sh;
sudo bash /home/$USER/Downloads/SECOND.sh;
```


* أنشئ ملفين:

الأول يحتوي على عمود لأصحاب العناوين،

والثاني يحتوي على عمود لعناوين البريد الإلكتروني

ثم أدخلهما مع استبدال حقول الوسائط في النمط:

```bash
/home/$USER/Downloads/batchproduction_ofTHIRD.sh <path/to/and/ownerfile.txt> <path/to/and/addressfile.txt>;
```

يسمح لك هذا البرنامج النصي الأخير بإدخال قوائم عناوين البريد الإلكتروني وأصحابها، مما ينتج عنه برامج نصية bash منفصلة لكل عنوان بريد إلكتروني.

```bash
 bash /home/$USER/Downloads/FIRST.sh
```

* عند تشغيل البرامج النصية المنفصلة، ​​سيتم إنشاء مفاتيح GPG وتخزينها على مفتاح USB مشفر باستخدام LUKS. هام: توصي OpenGPG باستخدام أي مفاتيح خاصة غير مشفرة ذاتيًا ثم حذفها فورًا لتجنب تخزين معلومات حساسة.

```bash
 bash /home/$USER/Downloads/SECOND.sh
```

الخطوة الأخيرة: فصل جهاز USB بأمان

بعد إتمام جميع الخطوات بنجاح، تأكد من فصل جهاز USB LUKS بأمان لتجنب أي مشاكل محتملة عند إعادة توصيله:

```bash
sudo umount "/mnt/usb_gpg"
sudo cryptsetup luksClose myusb_key
```

وإلا فقد تواجه صعوبات عند توصيل جهازك في المرة القادمة.

باتباع هذه الخطوات، يمكنك إنشاء وتخزين مفاتيح GPG الرئيسية والفرعية بشكل آمن على مفتاح USB مشفر باستخدام LUKS. تذكر دائمًا التعامل مع مفاتيحك الخاصة بعناية واتباع أفضل الممارسات الأمنية. إذا كانت لديك أي أسئلة أو مشاكل، يرجى الرجوع إلى الوثائق أو طلب المساعدة من المجتمع.
